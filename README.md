Pokémon Ascenso Dev Logs es una plataforma web donde se documenta el desarrollo del videojuego Pokémon Ascenso, un proyecto fan inspirado en los combates de la saga Pokémon.

La web funciona como diario de desarrollo + portfolio, mostrando la evolución del proyecto a través de versiones, mejoras técnicas y decisiones de diseño.

# 🚀 Demo (Netlify)
🔗 **Ver proyecto online:** https://pokemonascensodevlogs.netlify.app

# ⚙️ Características
- Sistema de devlogs organizados por versiones
- Estructura preparada para contenido multimedia (imágenes, vídeos)
- Secciones diferenciadas: home, galería, destacados y versiones
- Arquitectura escalable para añadir nuevos logs fácilmente

# 🛠️ Tecnologías utilizadas
- React + Vite
- JavaScript (ES6+)
- CSS3 (custom + theming)
- Netlify (deploy)
- GitHub (versionado)
- Arquitectura modular de componentes
