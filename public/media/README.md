# Media del devlog

Usa esta carpeta para guardar los materiales visuales del sitio.

## Estructura recomendada

- `home/hero/`: imagenes del hero de portada
- `home/hero-video/`: video de fondo del hero de la Home
- `home/captures/`: galeria visual de la portada
- `brand/logo/`: logo del juego
- `devlogs/hero-video/`: video de fondo del hero de Devlogs
- `devlogs/archive/`: captura de la carpeta con las 200+ versiones internas
- `versions/`: capturas o videos asociados a versiones concretas

## Convencion de nombres

Usa nombres numericos:

- `01.png`
- `02.jpg`
- `03.webp`
- `04.mp4`

La web intentara cargarlos automaticamente sin tocar el codigo.

## Ejemplos

- `public/media/home/hero/01.png`
- `public/media/home/hero-video/01.mp4`
- `public/media/home/captures/01.png`
- `public/media/brand/logo/01.png`
- `public/media/devlogs/hero-video/01.mp4`
- `public/media/devlogs/archive/01.png`
- `public/media/versions/0.8.0/01.png`
- `public/media/versions/0.8.0/02.mp4`
